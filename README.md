# Python 3
## Packages

- html2text
- pysrt
- webvtt-py

```
pip3 install html2text pysrt webvtt-py
```

# Synopsis

./vtt-to-srt FILE...

# Description

The command accepts one or more files as arguments.
For each _.vtt_, a _.srt_ will be generated in the same folder.
Any other extension is ignored.
